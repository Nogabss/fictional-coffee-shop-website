# Coffee Shop Website

A simple coffee shop website developed using pure HTML, CSS and JavaScript.

It simulates a small café system with pages for home, menu and order, allowing users to view products and place simple orders.

## 🚀 Technologies Used
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

## 📋 Features
- Home page with coffee shop presentation  
- Menu page displaying products  
- Order page with form  
- Responsive layout  
- Image gallery  
- Basic interactivity with JavaScript  

## ▶️ How to Run
1. Download or clone this repository  
2. Open the `index.html` file in your browser  
3. Navigate through the menu and order pages

- Styling layouts using CSS  
- DOM manipulation with JavaScript
